# Shopping Cart with React-Redux

Simple Shopping Cart with React-Redux

## What things I made use of:
- HTML5/CSS3/
- React - Redux
- React-Icons
- Google Fonts

## Features
- Add Products to cart
- Remove Cart Items from Cart
- Handle Total Price
- Clear Cart 
- LocalStorage
- Increase & Decrease Amount Of Items in Cart
